# Realtime Action Response Robot

## 프로젝트 개요
YOLO와 MediaPipe를 활용하여 사람의 행동을 실시간으로 분석하고, Raspberry Pi와 STM32를 연동해 상황에 맞는 대응 동작을 수행하는 로봇 시스템입니다.

## 사용 기술
- Raspberry Pi 5
- STM32
- Python
- YOLO
- MediaPipe
- OpenCV
- GPIO Parallel Communication

## 주요 기능
- 실시간 객체 탐지
- 행동 분석 및 자세 보정
- 상황별 대응 동작 수행
- 웹 기반 모니터링
- Raspberry Pi - STM32 연동 제어

## 내가 한 역할
- 행동 인식 파이프라인 구현
- YOLO 기반 탐지 모델 적용 및 튜닝
- MediaPipe 기반 자세 분석 보정
- Raspberry Pi와 STM32 간 통신 설계
- 상황별 로봇 제어 로직 구현

## 결과
- 실시간 행동 분석 및 대응 동작 구현
- 객체 탐지와 자세 분석을 결합한 인식 성능 향상
- 영상 모니터링과 로봇 제어를 통합한 시스템 구성 완료

## 개선 사항
- 행동 분류 정확도 향상
- 통신 안정성 개선
- 복잡한 환경에서의 추적 성능 고도화
